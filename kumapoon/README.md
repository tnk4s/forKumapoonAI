# kumapoon

Jump Kingっぽい何か

Getting Started
===============

poetryをインストールします。
[poetry](https://python-poetry.org/docs/)のページを参考に、自分の環境に合った方法でインストールしてください。

```
curl -sSL https://install.python-poetry.org | python -
```

kumapoonを使えるようにするため、installします。

```
poetry install
```

installできたら、ゲームを起動しましょう。

```
poetry run python src/main.py
```
